# CHANGE: 翻译助手测试覆盖增强

## Motivation

项目当前仅有单元测试。README 评估中集成测试、E2E 测试、性能测试、覆盖率报告全部待实现。

## Scope

- In:
  - 集成测试（翻译 API 完整调用链）
  - E2E 测试（前端到后端全链路）
  - 性能测试（并发请求 + 响应时间基准）
  - pytest-cov 覆盖率报告配置
- Out:
  - 负载测试工具部署（JMeter/K6）

## Risks

- E2E 测试需要服务运行，依赖 CI 环境配置
- 性能测试基准需在稳定环境中校准

## Success Criteria

- [x] 集成测试覆盖 6 个翻译方向
- [x] E2E 测试通过前端发送请求到后端
- [x] 性能测试 P95 < 3s
- [x] 覆盖率报告 > 80%
